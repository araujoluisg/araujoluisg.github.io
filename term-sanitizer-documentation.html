Term Sanitizer by Luís Araújo

Este modelo de variável JavaScript personalizada no GTM tem a função de sanitizar um termo de pesquisa, aplicando diferentes transformações conforme as opções definidas pelo usuário.

Funcionamento

Recebimento de parâmetros:

  • originalTerm: termo original a ser tratado.
  • removeSpaces (opcional, padrão: false): define se os espaços serão removidos.
  • separatorType (opcional, padrão: "-"): caractere separador usado ao remover espaços.
  • caseOption (opcional, padrão: "lowercase"): define se o texto será convertido para minúsculas ou maiúsculas.

Sanitização inicial:

  • Se o originalTerm não for uma string, retorna uma string vazia.
  • Remove espaços extras no início e no final do termo.

Remoção de acentos e caracteres especiais:

  • Substitui caracteres acentuados por suas versões sem acento (ex.: á → a).
  • Troca _ e - por espaços.

Filtragem de caracteres inválidos:

  • Mantém apenas letras (A-Z e a-z), números (0-9) e espaços.
  • Remove espaços consecutivos.

Ajustes finais:

  • Se removeSpaces for true, substitui espaços pelo separatorType definido.
  • Converte o texto para minúsculas ou maiúsculas conforme caseOption.

Exemplo de Uso

Entrada:

data = {
                  originalTerm: "Olá_QCâRábõM! É incrívêl, e_tem térMôs_trâTADõs",
                  removeSpaces: true,
                  separatorType: "-",
                  caseOption: "lowercase"
                };

Saída:

"ola-qcarabom-e-incrivel-e-tem-termos-tratados"

Essa variável é útil para garantir que termos capturados pelo GTM estejam padronizados antes de serem enviados para ferramentas de análise, como Google Analytics ou DataLayer.
